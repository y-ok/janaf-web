# Janaf-web

## Start janaf-web

```cmd
$ git clone https://github.com/y-ok/janaf-web.git
$ cd janaf-web
$ sh docker-start.sh product
```

## Access janaf-web
  http://192.168.99.100:8080/elements  

## Stopping janaf-web

```cmd
$ sh docker-stop.sh
```

## Remove docker-image, docker-process

```cmd
$ sh docker-remove-all.sh
```
